# pokemon_ai

A voice-powered AI communication skills coach for students, built with React + Vite.

## Features
- 🎤 Voice input — speak to practice, AI listens in real time
- 🧠 AI Communication Coach (Aria) — gives feedback on grammar, clarity, confidence
- 🗣️ Practice modes — Self-intro, Mock interviews, Pronunciation, Debate, Presentations
- 💬 Real-time chat — orange-themed premium UI inspired by modern AI apps
- 📱 Fully responsive — mobile phone UI + full-screen desktop

## Tech Stack
- React 18 + Vite
- Lucide React icons
- OpenRouter AI API (nvidia/nemotron model)
- Web Speech API for voice recognition
- Vanilla CSS with custom animations

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel
This project is configured for Vercel. Connect your GitHub repo to Vercel and it will auto-deploy.
